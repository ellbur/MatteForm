
matteform
=========

``matteform`` is a form-processing library for the `Lift Framework
<http://liftweb.net/>`_

Motivation
----------

 - Avoids reflection
   
 - Fails-fast

 - All rendering done in HTML; none of it in code

Building
--------

The build is done using `SBT <https://github.com/harrah/xsbt>`_ version 0.11.0::

    sbt package
